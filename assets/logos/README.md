# Logo Placeholders

Folder ini berisi placeholder untuk logo keterampilan. Ganti file-file berikut dengan logo asli:

## File yang Diperlukan:

### Bahasa Pemrograman:
- `python-logo.png` - Logo Python (ukuran yang disarankan: 128x128px atau lebih)
- `java-logo.png` - Logo Java
- `javascript-logo.png` - Logo JavaScript
- `sql-logo.png` - Logo SQL/Database

### Framework & Library:
- `react-logo.png` - Logo React.js
- `nodejs-logo.png` - Logo Node.js

### Tools & Version Control:
- `git-logo.png` - Logo Git
- `docker-logo.png` - Logo Docker
- `figma-logo.png` - Logo Figma
- `vscode-logo.png` - Logo VS Code
- `postman-logo.png` - Logo Postman
- `tableau-logo.png` - Logo Tableau

## Format yang Disarankan:
- Format: PNG dengan background transparan
- Ukuran: 128x128px atau 256x256px
- Pastikan logo berbentuk persegi atau dapat dipotong menjadi persegi

## Sumber Logo:
Anda bisa mendapatkan logo resmi dari:
- Website resmi masing-masing tools
- GitHub repository resmi
- Icon libraries seperti DevIcons atau Simple Icons
- Google Images (pastikan menggunakan logo resmi dan berkualitas tinggi)

## Alternatif Sementara:
Jika belum memiliki logo, Anda bisa menggunakan:
- Placeholder image generators
- Simple colored squares dengan inisial
- SVG icons dari icon libraries

Setelah menambahkan logo, hapus file README.md ini.
